# Tutor — Интеллектуальная система записи к репетитору

Современный, высококонверсионный Landing Page для частного преподавателя английского языка с интегрированной системой моментальных уведомлений в Telegram.

## Особенности проекта

* **Коммерческий дизайн:** Чистый, профессиональный интерфейс, вызывающий доверие у клиентов.
* **Mobile-First:** Полная адаптация под любые устройства (от 320px до Ultra-Wide мониторов).
* **Интерактивный UI:** Живой выбор программ обучения с автоматическим обновлением формы записи.
* **Fullstack-интеграция:** Связка фронтенда с кастомным API на FastAPI для обработки заявок.
* **Мгновенные уведомления:** Все заявки прилетают владельцу напрямую в Telegram в течение 1 секунды.

## Технологический стек

### Frontend:
* **HTML5 & CSS3:** Семантическая верстка, современные CSS-переменные, Flexbox/Grid.
* **Vanilla JavaScript:** Асинхронные запросы (Fetch API), обработка состояний формы, Intersection Observer для анимаций.
* **GitHub Pages:** Хостинг для статического контента.

### Backend:
* **Python (FastAPI):** Высокопроизводительный асинхронный сервер для обработки заявок.
* **Telegram Bot API:** Система автоматического оповещения.
* **Render:** Хостинг бэкенд-части.

## Как это работает

1. Пользователь заходит на сайт и выбирает интересующую его программу.
2. JavaScript перехватывает клик, визуально выделяет карту и подставляет значение в скрытое поле формы.
3. При отправке формы данные улетают на FastAPI-сервер.
4. Бэкенд валидирует данные и через Telegram-бота отправляет структурированное сообщение администратору.

## Структура проекта

* `/index.html` — главная страница.
* `/css/style.css` — стили проекта (адаптив + анимации).
* `/js/script.js` — логика взаимодействия и отправка данных.
